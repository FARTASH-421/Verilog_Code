This example is of the UVVM (Universal VHDL Verification Methodology).
Three examples exist. bitvis_uart, bitvis_irqc, and bitvis_scoreboard.
To run the example change your directory to
 *  cd <install_dir>/examples/vhdl/uvvm/<example>script
 *  invoke vsim
 *  execute "do compile_all_and_simulate.do"

Result transcripts are located in the directory examples_results with a transcript
for each test.
